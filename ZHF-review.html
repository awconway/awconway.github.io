2019 Summer Work Study Program



Zero heat flux thermometry: A systematic review and meta-analysis


Goals

  • Learn how to use Covidence to manage study selection, data extraction and risk of bias assessment
  • Learn how to use the RStudio IDE (or RStudio Cloud)
  • Learn how to use version control with RStudio
  • Learn how to use R to perform meta-analysis of Bland-Altman method comparison studies
  • Prepare a manuscript for publication

Activities

Complete the following Datacamp courses

  • Introduction to R (4 hours)
  • Working with the RStudio IDE (Part 1) (3 hours)
  • Chapter 2 - Version Control and Chapter 3 - Reporting of Working with the RStudio IDE (2 hours)
  • Reporting with R Markdown (3 hours)
  • Introduction to the Tidyverse (4 hours)

Set-up R and RStudio

  • Follow the instructions here for RStudio Cloud

Complete the following Cochrane courses

  • Introduction to conducting systematic reviews
  • Searching for studies
  • Selecting studies and collecting data
  • Introduction to study quality and risk of bias

Systematic review

  • Write a protocol for the systematic review and register at PROSPERO
  • Devise search strategy
  • Undertake literature search of databases and import results into Covidence
  • Screen titles and abstracts
  • Review full text of potentially relevant articles for inclusion
  • Extract data and perform risk of bias assessment using QADAS-II
  • Export data from Covidence and import to RStudio project
  • Prepare a report using R Markdown to display the results and statistical code used for the analyses undertaken
  • Commit and push to Github as progress is made
  • Prepare a manuscript for publication
