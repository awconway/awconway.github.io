2019 Undergraduate Student Summer Research Program



Transcutaneous carbon dioxide monitoring during procedural sedation in the cardiac catheterisation laboratory


Goals

  • Learn how to use R to tidy, transform and visualise data, as well as some basic modelling
  • Learn how to use the RStudio IDE (or RStudio Cloud)
  • Learn how to use version control with RStudio
  • Prepare a manuscript for publication

Activities

View the following RStudio instruction videos

  • Writing code in RStudio (0.5 hours)
  • Projects in RStudio (0.5 hours)
  • Github and RStudio
  • Getting Started with R Markdown (1 hour)

Complete the following Datacamp courses

  • Introduction to R (4 hours)
  • Introduction to the Tidyverse (4 hours)
  • Data viz with ggplot (Part 1) (5 hours)
  • Correlation and Regression (4 hours)

Set-up R and RStudio

  • Follow the instructions here for RStudio Cloud

Analysis and Report Writing

  • Undertake descriptive analyses of changes in transcutaneous carbon dioxide concentrations during sedation
  • Perform multivariable modelling of post-sedation mean and peak transcutaneous carbon dioxide concentrations
  • Create data vizualisations
  • Prepare the report using R Markdown to display the results and statistical code used for the analyses undertaken
  • Commit and push to Github early and often
